HTML Basics
HTML (HyperText Markup Language) adalah blok bangunan dasar web. HTML digunakan untuk membuat dan mengatur konten di halaman web, seperti paragraf, judul, daftar, tautan, dan lainnya. HTML menggunakan tag untuk mendefinisikan berbagai elemen halaman, dan setiap tag menggambarkan bagian yang berbeda dari konten.
